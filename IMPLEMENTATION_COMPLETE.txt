🎯 LOGIN & TESTING

To Test as Regular User:
1. Username: john (or any name)
2. Role: Select "👤 Regular User"
3. Click "🔓 Sign In"
4. Create/edit/delete notes
5. Notice no "Admin Logs" button
6. Click "🚪 Logout" in top-right

To Test as Administrator:
1. Username: admin (or adm)
2. Role: Select "🔐 Administrator"
3. Click "🔓 Sign In"
4. Same note CRUD as user
5. See "🔍 Admin Logs" button at bottom
6. Expand logs to see all activities
7. Filter, export, and clear logs
8. Click "🚪 Logout" in top-right


📦 INSTALLATION & USAGE
═════════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies
$ npm install

This installs:
├─ tailwindcss          (Main CSS framework)
├─ postcss              (CSS processor)
├─ autoprefixer         (Vendor prefixes)
└─ @tailwindcss/forms   (Form styling plugin)

Step 2: Start Development Server
$ npm run dev

App opens at: http://localhost:5173

Step 3: Build for Production
$ npm run build

Creates optimized build in dist/ folder
