   User Input → Validation → API Call → Log "CREATE" → Reload Notes

   Page Load → Fetch Notes → Update State → Log "READ" → Display Grid

   Click Edit → Load Form → User Changes → API Call → Log "UPDATE" → Reload

DELETE NOTE:
   Click Delete → Show Confirmation → API Call → Log "DELETE" → Reload

SEARCH:
   User Types → Filter State → Display Matches (Real-time)

ADMIN ACTIONS:
   View Logs → Filter by Type → Export JSON → Clear with Confirmation
